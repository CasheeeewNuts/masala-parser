single

    • Defined in masala-parser.d.ts:374

    Map the response value as the first Tuple element.

    const parser = C.char('a')
   // 'b' is dropped, but we still have a Tuple
  .then(C.char('b').drop())
  .single();

test.equal(value, 'a');

    WARNING: This may change and throw an exception if it's not a single value. first() may appear.
