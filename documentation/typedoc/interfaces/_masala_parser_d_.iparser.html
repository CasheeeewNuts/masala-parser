Interface IParser<T>

Parsers are most of the time made by combination of Parsers given by CharBundle (C**), NumberBundle (N), and/or FlowBundle (F**). You can also create a custom Parser from scratch. T is the type of the Response

occurrence

  • occurrence(n: number): TupleParser<T>
    • Defined in masala-parser.d.ts:607

    Search for next n occurrences. TupleParser.occurence() will continue to build one larger Tuple

    let parser = C.char('a').then (C.char('b')).occurrence(3);
let resp = parser.parse(Streams.ofString('ababab'));
